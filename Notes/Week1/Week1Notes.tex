\textbf{Solutions to Systems of Linear Equations}\\
\textsf{A unique solution(Consistent)}\\
$x - 2y = -1$\\
$x = 2y-1$\\
$-2y+1+3y = 3$\\
$y = 2$\\
$x - 4 = -1$\\
$x = 3$\\
$(x, y) = (3, 2)$\\
\textsf{Infinitely many solutions(Consistent)}\\
$x - 2y = -1$\\
$-x+2y=3$\\
$x = 2y-1$\\
$-2y+1+2y=3$\\
$0y=2$\\
\textsf{No solutions(Inconsistent)}\\
$x-2y=-1$\\
$-x+2y=1$\\
$x=2y-1$\\
$-2y+1+2y=1$\\
$0y=0$\\

\textbf{Linear Equations}\\
\textsf{Method 1}\\
1) $x_1 - 2x_2 = -1$\\
2) $-x_1 + 3x_2 = 3$\\
Rewrite (1) as:\\
3) $x_1 = -1 + 2x_2$\\
Sub (3) into (2):\\
$-(-1+2x_2) + 3x_2 = 3$\\
$1 - 2x_2 + 3x_2 +2 = 3$\\
$1 + x_2 = 3$\\
$x_2 = 3 - 1$\\
4) $x_2 = 2$\\
Sub (4) into (1):\\
$x_1 - 2(2) = -1$\\
$x_1 - 4 = -1$\\
$x_1 = -1 + 4$\\
$x_1 = 3$\\
\textsf{Method 2}\\
1) $x_1 - 2x_2 + x_3 = 0$\\
2) $2x_2 - 8x_3 = 8$\\
3) $-4x_2 + 5x_2 + x_3 = -9$\\
Multiply (1) by 4\\
$4x_1 - 8x_2 + 4x_3 = 0$\\
$-4x_1 + 5x_2 + x_3 = -9$\\
Add both\\
$-3x_2 + 5x_3 = -9$\\
$2x_2 - 8x_3 = 8$\\
Make the coefficient the same(By multiplication)\\
$-6x_2 + 10x_3 = -18$\\
$6x_2 - 24x_3 = 24$\\
Addition\\
$-14x_3 = 6$\\
$x_3 = \frac{-6}{14}$\\
